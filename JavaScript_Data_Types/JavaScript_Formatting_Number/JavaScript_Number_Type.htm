JavaScript Number Types

JavaScript numbers can be written in several formats:

  • Signed Integer: -20
  • Unsigned Integer: 40
  • Floating Point: 23.34
  • Double (JS uses double precision): 340.99
  • Exponent Form: 2e3 (2 × 10³ = 2000)
  • Binary: 0b0101 (5)
  • Hexadecimal: 0x7340
  • Octal: 0o734
  • BigInt: 987654676896n
